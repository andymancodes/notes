# Gamma Function

For positive integer n,
$$\Gamma (n)=(n-1)!$$
Generally,
$$\Gamma (t)=\int _{0}^{\infty }x^{t-1}e^{-x}\,dx$$
Gamma Function is an extension of the factorial function.
